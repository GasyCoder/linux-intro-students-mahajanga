# Exercice 1 pratiques — Manipulation de fichiers et arborescence

---

## Partie A — Création de structure

1. Créer l’arborescence suivante :

projet-linux/
    src/
    config/
    logs/

2. Vérifier votre position avec :
pwd

3. Vérifier la structure avec :
ls -R

---

## Partie B — Création de fichiers

Dans le dossier src :

- créer app.js
- créer database.js

Dans config :

- créer app.conf

Dans logs :

- créer app.log

---

## Partie C — Manipulations

1. Copier app.js vers app_backup.js
2. Renommer database.js en db.js
3. Supprimer app_backup.js

---

## Partie D — Rédaction de rapport

1. Générer un fichier rapport.txt contenant :

- Le résultat de ls -l
- Une ligne ajoutée avec :
  echo "Projet initialisé avec succès"

2. Vérifier le contenu avec :
cat rapport.txt
